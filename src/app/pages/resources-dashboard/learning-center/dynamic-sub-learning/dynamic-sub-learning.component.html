What is the Property Tax Digital Toolkit?
This toolkit is a one-stop-shop for all your needs for reforming property taxes in your State/ULB. It provides reform recommendations based on best-practices implemented nationally and internationally and provides a step-wise implementation guide that you can easily follow to take immediate action. Reform recommendations in each stage of the lifecycle of property tax have been framed as responses to issues and implementation challenges, sharply addressing WHAT NEEDS to BE DONE and HOW CAN IT BE DONE.
