Welcome to the State Dashboard

This dashboard shows revenue and expenditure data from FY 2019–20 to FY 2022–23.

The data is based on submissions from various cities, with extrapolated values where direct data wasn’t available.

The default year has been fixed to FY 2019–20.

With this dashboard, you can:

  • Compare the progress of one city against another
  • Benchmark states against other states
  • Understand trends and performance for chosen years
